# French Mini-Lesson: Describing "Le taux de [nom] [verbe]"

---

## Core Structure / Pattern
**Le taux de [nom] [verbe] [complément]**   

---

## Key Vocabulary

### Nouns
- chômage
- réussite 
- mortalité
- récidive
- vaccination
- participation
- inflation
- croissance
- absences 
- accidents

### Verbs

#### Adjectives & Expressions with "Être"
- élevé → Le taux de chômage est **élevé**.
- faible → Le taux d'intérêt est très **faible**.
- en hausse → Le taux d'inflation est **en hausse**.
- en baisse → Le taux de criminalité est **en baisse**.

#### Hausse / Increase & Speed Up
- augmenter → Le taux de participation **augmente** chaque année.
- accélérer → Le taux de croissance **accélère** rapidement.

#### Baisse / Decrease & Slow Down
- diminuer → Le taux de mortalité **diminue**.
- ralentir → Le taux d'inflation **ralentit**.
- reculer → Le taux de récidive **recule**.
- s'effondrer → Le taux de change **s'effondre** brutalement.

#### Stabilité & Variation
- rester stable → Le taux d’inflation **reste stable**.
- stagner → Le taux de chômage **stagne**.
- fluctuer → Le taux de change **fluctue** selon les saisons.

#### Point Spécifique
- atteindre → Le taux de réussite **atteint** 90%.
